Installation
------------

The ``corerl`` package is closed source and proprietary.
It may be installed in editable mode with the `rlcoretech/core-rl <https://github.com/rlcoretech/core-rl>`_ repository cloned locally.
To use corerl, first install it using pip:

.. code-block:: console

   (.venv) $ pip install -e .
